Campus Placement Portal - Enhanced (All features)
Features included:
- Admin panel (manage students, jobs, applications, approve/reject)
- Resume upload (students upload PDFs/DOCs)
- Company accounts (post/manage jobs)
- Job search & filters
- Email notifications (php mail placeholder)
- Student profile & photo upload
- Application status tracking (Pending/Approved/Rejected)
- Activity logs (login, post job, apply)
- Analytics dashboard (counts + Chart.js)
- Modern UI using Bootstrap 5 + custom CSS
Setup:
1. Extract to web root. Edit config.php with DB credentials.
2. Create uploads/resumes and uploads/avatars writable by web server.
3. Import db.sql into MySQL.
4. Visit /index.php
